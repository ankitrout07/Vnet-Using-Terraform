    Project Architecture & Tech Stack

    Fully automated, highly available infrastructure deployed on Azure

    🏗️

    Infrastructure as Code

    • Terraform: Core provisioning engine (v1.5.0+)
    • AzureRM Provider: Manages all Azure resources
    • Modular Design: Networking, AKS, App Gateway, Database, Redis, and Bastion isolated in reusable modules
    🕸️

    Networking & Security

    • VNet: Custom isolated 10.0.0.0/16 network
    • Subnet Tiers: Public, App, Data, Cache, Gateway, Bastion
    • App Gateway (WAF v2): Ingress controller with Web Application Firewall
    • Bastion: Secure RDP/SSH without public IPs on VMs
    ⚙️

    Compute & Workloads

    • Azure Kubernetes Service (AKS): Orchestration of application containers
    • Azure Container Registry (ACR): Private Docker image storage
    • NodeJS / Express: Real-time backend dashboard server
    • Socket.io: WebSocket communication for live metrics
    🗄️

    Data & State

    • PostgreSQL Flexible Server: Relational data storage
    • Azure Cache for Redis: High-speed data caching
    • Azure Blob Storage: Remote Terraform state backend for team collaboration
    🚀

    CI/CD Pipeline

    • GitHub Actions: Automated build and deployment
    • Idempotent Workflow: `terraform plan -detailed-exitcode` prevents empty applies
    • Content Hashing: Docker images tagged via SHA1 of source files
    🎨

    Frontend UI

    • Vanilla HTML/CSS/JS: Zero-dependency, lightweight interface
    • SVG + CSS Animations: Live network map with pulsing data flow
    • Glassmorphism: Modern, cyber-ops aesthetic
